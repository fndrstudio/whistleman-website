Audit all `<img>` tags across all HTML files and add descriptive alt text. Current state shows many empty `alt=""` attributes.

**Rules for alt text:**

1. **Content images** (photos, case study images, hero images):
   - 80-125 characters describing the image
   - Be specific: "Chef preparing pasta in restaurant kitchen" not "food photo"
   - Include relevant context: client name, location, what's happening

2. **Client logos**:
   - Include company name: `alt="Oeuf Amsterdam restaurant logo"`

3. **Decorative images** (icons, backgrounds, purely visual elements):
   - Keep `alt=""` empty - this tells screen readers to skip
   - Add `role="presentation"` if appropriate

4. **Navigation icons**:
   - Describe the action: `alt="Portfolio icon"` or `alt="Contact icon"`
   - If icon has adjacent text label, can use `alt=""` to avoid redundancy

**Examples of good alt text for this site:**

| Image | Good Alt Text |
|-------|--------------|
| Hero background | "Video production team filming on location at Amsterdam restaurant" |
| Oeuf case study hero | "Oeuf Amsterdam brunch restaurant interior with natural lighting" |
| Client logo carousel | "Burger 'n Shake restaurant logo" |
| Team/testimonial photo | "Lorenzo Isodoro, Chef at Linguini restaurant Amsterdam" |
| Portfolio card image | "Oeuf case study showing social media content creation results" |

**Do NOT:**
- Stuff keywords: `alt="video production social media marketing amsterdam agency content creation"`
- Be too generic: `alt="image"` or `alt="photo"`
- Duplicate the page title or caption already visible

**Process:**
1. For each HTML file, find all `<img>` tags
2. For `alt=""` (empty), determine if decorative or content
3. If content, add descriptive text based on the image filename and context
4. If decorative (arrows, icons with text labels), leave empty
